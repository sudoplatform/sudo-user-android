ApiErrorCode

enum class ApiErrorCode

Enum Values

IDENTITY_NOT_CONFIRMED

Identity was created but not confirmed because validation failed.

INVALID_CONFIG

Invalid configuration parameters were passed.

INVALID_INPUT

Invalid input parameters were passed.

NOT_AUTHORIZED

User is not authorized to perform the operation requested.

NOT_REGISTERED

Client is not registered.

NOT_SIGNED_IN

Client is not signed in.

ALREADY_REGISTERED

Client is already registered. This could result from calling one of the register API when the client was already registered.

SERVER_ERROR

An internal server error cause the API call to fail. The error is possibly transient and retrying at a later time may cause the call to complete successfully.

GRAPHQL_ERROR

GraphQL endpoint returned an error.

FATAL_ERROR

Unexpected error encountered. This could be a result of client or backend bug and unlikely to be user recoverable.
