DefaultSudoUserClient

class DefaultSudoUserClient : SudoUserClient

Default implementation of SudoUserClient interface.

Parameters

config - configuration parameters.

context - Android app context.

namespace - namespace to use for internal data and cryptographic keys. This should be unique per client per app to avoid name conflicts between multiple clients.

logger - logger to use for logging messages.

keyManager - custom KeyManagerInterface implementation. Mainly used for unit testing (optional)

identityProvider - custom identity provider. Mainly used for unit testing (optional).

apiClient - custom API client. Mainly used for unit testing (optional).

credentialsProvider - custom credentials provider. Mainly used for unit testing (optional).

authUI - custom auth UI. Mainly used for unit testing (optional).

idGenerator - custom ID generator. Mainly used for unit testing (optional).
