signInWithAuthenticationProvider

fun signInWithAuthenticationProvider(authenticationProvider: AuthenticationProvider, callback: (SignInResult) -> Unit): Unit

Sign into the backend with an external authentication provider. Caller must implement AuthenticationProvider protocol to return the appropriate authentication token associated with the external identity registered with registerWithAuthenticationProvider.

Parameters

authenticationProvider - authentication provider that provides the authentication token.

callback - callback for returning sign in result containing ID, access and refresh token or error.
